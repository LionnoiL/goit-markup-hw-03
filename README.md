# WEBSTUDIO

## Description

The WEBSTUDIO project is the third homework assignment on HTML markup and CSS styling with Flexbox layout as part of
the GoIT Fullstack program. The project demonstrates the creation of a semantic
HTML structure for a digital solutions agency website with modular CSS styling and modern Flexbox layout techniques.

## Project Structure

```
goit-markup-hw-03/
├── index.html              # Main HTML file with markup
├── css/
│   ├── main.css            # Main CSS file (imports all other stylesheets)
│   ├── common.css          # Common styles (body, headings, links, buttons, etc.)
│   ├── header.css          # Header styling with Flexbox
│   ├── hero.css            # Hero section styling
│   ├── features.css        # Features section styling with Flexbox layout
│   ├── team.css            # Team section styling with Flexbox layout
│   ├── portfolio.css       # Portfolio section styling with Flexbox layout
│   └── footer.css          # Footer styling with Flexbox

```

## Website Content

### Header

- WEBSTUDIO company logo
- Navigation menu (Features, Team, Portfolio)
- Contact information (email and phone)
- Flexbox layout for horizontal alignment

### Main

- **Hero section** — main heading with call-to-action button and dark background
- **Features section** — list of key company advantages displayed in a Flexbox grid
- **Team section** — team members presentation with Flexbox layout and cards
- **Portfolio section** — showcase of company projects with Flexbox grid layout

### Footer

- Company logo
- Business description
- Flexbox layout for content alignment
- Dark background styling

## Technologies

- **HTML5** — semantic markup
- **CSS3** — modular styling with Flexbox layout
- **Flexbox** — modern layout technique for responsive design
- **Modern Normalize** — CSS reset library
- **Google Fonts** — Roboto and Raleway font families
- Proper document structure
- Use of semantic tags (`<header>`, `<main>`, `<footer>`, `<section>`, `<nav>`, `<address>`)

## Key Features Compared to HW-02

- **Flexbox Layouts** — all sections use CSS Flexbox for proper alignment and distribution
- **Container-based Design** — proper use of flex containers and flex items
- **Responsive Flexibility** — better foundation for future responsive design
- **Improved Spacing** — proper gaps and margins using Flexbox properties
- **Better Code Organization** — logical structure for managing complex layouts

## Project Goal

This project is designed to practice the following skills:

- Creating semantically correct HTML markup
- Organizing content into logical sections
- Using anchor links for navigation
- Proper embedding of media elements
- Applying CSS styling to HTML elements
- Working with typography and colors
- **Implementing Flexbox layouts**
- **Creating flexible and responsive component layouts**
- **Using flex properties (display, justify-content, align-items, gap, etc.)**
- **Understanding flex direction and wrapping**
- **Organizing CSS with modular architecture**
- Advanced layout techniques with modern CSS

## Running the Project

Open the `index.html` file in a web browser to view the website.
